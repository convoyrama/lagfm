# LAG.FM Radio Suite
> **By the community, for the community.**

LAG.FM is a cross-platform radio client for simulator drivers (ETS2 / ATS). It provides audio streaming and real-time telemetry integration as a simple utility to help on the road.

## 📻 Overview
This tool focuses on functional immersion:
- **Live Radio:** Tune into community stations and custom streams.
- **Real-time Telemetry:** Connect to your PC via LLP to see speed, fuel, and gear data.
- **Mobile Support:** Native background resilience for stable streaming and data flow.
- **Simple UI:** Designed as a functional radio unit for a clean driving experience.

## 🚀 Download & Installation

### Android
Download the latest APK from the **Releases** section.
1. Go to [Releases](https://github.com/CONVOYRAMA/lagfm/releases).
2. Download `LAGFM_vX.X.X.apk`.
3. Install and run.

### Desktop (Windows / Linux)
Installers are available in the **Releases** section.
- **Windows:** Download the `.msi` installer.
- **Linux:** Use the `.AppImage` file.
