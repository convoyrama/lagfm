# LAGFM Desktop (Technical Docs)
> **Platform:** Windows / Linux | **Stack:** Tauri v2 + React

Source code for the desktop radio client.

## 🛠️ Development
- **Setup:** `npm install`
- **Dev Mode:** `npm run tauri dev`
- **Build Installers:** `npm run tauri build`

## ⚙️ Specs
- **Window:** 300x600 px, transparent and "always on top" support.
- **Audio:** HTML5 Audio engine with optimized CSP for external streams.
- **Builds:** Automated via GitHub Actions for `.msi` and `.AppImage`.
